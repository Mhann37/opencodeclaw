# Second Brain

This is the structured layer that sits between raw daily notes and long-term curated memory.

## Intent
- Keep MEMORY.md small and durable
- Keep daily notes lightweight and raw
- Keep projects, tasks, decisions, and references organized

## Structure
- `projects/` — active and recent project records
- `tasks/` — inbox and structured next actions
- `decisions/` — important decisions and constraints
- `areas/` — stable domains of responsibility
- `reference/` — reusable notes, research, and strategy docs
