Brief not sending?
  → Check environment variables: echo $TELEGRAM_BOT_TOKEN
  → Check logs: tail -f /var/log/health-brief.log

Data not loading?
  → Verify StrengthInsight file exists:
    find ~/research -name "*strength*.json" | head -1
  → Verify WHOOP cache:
    cat ~/.openclaw/workspace/.whoop-cache.json

Files corrupted?
  → Clean and rebuild:
    rm -f ~/.openclaw/workspace/.whoop-cache.json
    # Then re-upload StrengthInsight JSON
