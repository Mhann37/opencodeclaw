# Fraud Risk Assessment Tool - Real Estate Edition

A no-code assessment tool that helps real estate agents identify their top fraud risks in 2-3 minutes.

## Project Overview

- **User journey:** Fill 6-question quiz → Get personalized top-3 fraud risks → Download $299 playbook
- **Tech stack:** Typeform (quiz) + Landing page (HTML) + Gumroad (playbook)
- **Timeline:** 2-3 hours to launch
